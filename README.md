# todo 

Manage your todo list.

## installation

### mac os

```
brew install ynfeng/tab/go-todo
```

## basic usage

add item
```
todo add example

example added.
```

complete item

```
todo done 1

example done.
```

list items 

```
todo list 

1. Item
```

```
todo list -a

1. [Done] example
2. other

```
